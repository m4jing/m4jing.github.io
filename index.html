body 标记

  • bgcolor: 背景颜色:bgcolor="#fff"
  • background: 背景图片:图片在颜色层之上:background="html-meta.png"
  • text: 文本颜色
  • link: 链接文字颜色
  • vlink: 访问过的文本颜色
  • alink: 激活的链接文本颜色
  • leftmargin: 左边界
  • rightmargin: 右边界
  • topmargin: 上边界
  • bottommargin: 下边界

  • 默认情况下：左／上／右／下边界均默认为10px;

font 标记

  • color: 颜色
  • size: 字号
  • face: 字体

  • 网页中的文本 字体／字号一般用CSS控制

字符格式

  • 加粗：b
  • 倾斜：i
  • 加强语气(加粗)：strong
  • 加强语气(倾斜)：em
  • 下划线：u
  • 删除线：s
  • 上标：sup AB
  • 下标：sub AB

  • 同样是加粗，一般来说用 <strong> 而不用 <b> :strong对于搜索引擎有积极作用(strong更有利于 SEO)，而 b 则对于搜索引擎没有反应

段落标记 p

  • <p align="..">..</p>
  • align: left(左对齐，默认)/center居中/right右对齐

  • 如果不用 p 标记，几段文字放在一起，即使源文件中已经分段，但是最终呈现给用户的仍然是3段文件粘在一起…使用 p 标记，可以使文章结构更清晰。

段落标题 hx

  • <hx align="..">..</hx>
  • x取值：1-6；hx内的文字会加粗显示；hx针对段落，font针对任意文本
  • br：换行(换行不换段)
    • 请仅仅把标题标签用于标题文本。不要仅仅为了产生粗体文本而使用它们。请使用其它标签或 CSS 代替。搜索引擎使用标题为您的网页的结构和内容编制索引。因为用户可以通过标题来快速浏览您的网页，所以用标题来呈现文档结构是很重要的。
    浏览器会自动地在标题的前后添加空行.
    默认情况下，HTML 会自动地在块级元素前后添加一个额外的空行，比如段落、标题元素前后。

    title的作用：除了给 user 提供信息，还给搜索引擎提供抓取。可以在 title 中写上 keywords。还有 meta 信息可供使用。

HTML 中的特殊标记

  • 定义一个块引用：使用文本缩进<blockquote>..</blockquote>
    • blockquote 标记中的所有文本都会从常规文本中分离出来，经常会在左右两边缩进(增加外边距)，而且有时候会使用斜体。也就是说，块引用拥有它们自己的空间。
  • 属性名称：cite=url 被引用的地址；这个属性有利于 SEO
  • 居中标记：<center>..</center> (deprecated)
  • 预格式化标记：<pre>..</pre>：显示已经格式化好的文字。不加此标记，HTML 会忽略所有空格和制表符
    • 当显示页面时，浏览器会移除源代码中多余的空格和空行。所有连续的空格或空行都会被算作一个空格。需要注意的是，HTML 代码中的所有连续的空行（换行）也被显示为一个空格。

列表 ul，ol，dl

列表标记可以创建无序／有序／定义列表。还可以在一种列表中嵌套另一种列表。便于概括显示一系列相关内容

  • 无序：<ul>..</ul>
    •         <ul type="项目符号类型">
            <li type="项目符号类型">..</li>
            <li>...</li>
        </ul>

    项目符号类型：disc(default)／circle／square

  • 有序：<ol>..</ol>
    •         <ol start="列表起点" type="项目符号类型">
            <li>..</li>
            <li>...</li>
        </ol>

    项目符号类型：1(default)／a／A／i／I

  • 定义：<dl>..</dl>
    •         <dl>
            <dt>..</dt>
            <dd>...</dd>
            <dt>..</dt>
            <dd>...</dd>
        </dl>

    dt 表示一个项目；dd 表示这个项目的具体描述

图片标记 img

把图片的 img 代码插入到一堆文字中，来实现“图文混排”，并设置好 hspace 和 vspace 来控制图片与文字之间的空隙。
